## 学习建议

1. **按顺序学习**：教程是按照从简单到复杂的顺序设计的，建议按顺序学习
2. **动手实践**：每个教程都包含代码，建议跟随教程自己实现一遍
3. **理解原理**：不仅要会写代码，更要理解背后的原理
4. **参考源码**：遇到问题时，可以参考参考代码中的对应分支

## 环境配置

在开始学习之前，请确保您已经：

1. ✅ 配置好开发环境（参考环境配置文档）
2. ✅ 安装了 Rust nightly 工具链
3. ✅ 安装了 `x86_64-unknown-none` 目标
4. ✅ 安装了 `bootimage` 工具
5. ✅ 安装了 QEMU
