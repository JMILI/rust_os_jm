# Rust OS 项目目录

本目录包含基于教程逐步实现的操作系统项目。

## 📁 项目列表

### os_01 - 独立式 Rust 二进制程序 ✅

- **状态**: ✅ 已完成并可以编译
- **教程**: [01_独立式Rust二进制程序.md](../tutorial/src/01_独立式Rust二进制程序.md)
- **说明**: 创建一个不链接标准库的独立式 Rust 二进制程序

**快速开始**:
```bash
cd os_01
./build.sh
# 或
cargo rustc -- -Clink-arg=-nostartfiles
```
